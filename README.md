# PetPal 🐾

A responsive landing page for a fictional pet care booking service, built as a practice project from [The Odin Project](https://www.theodinproject.com/) foundations curriculum.

## Live Demo

[View the site](https://johnshoko-byte.github.io/Odin_landing_page/)

## About the Project

PetPal is a pet care booking service landing page built with plain HTML and CSS. The goal was to recreate a provided design template while applying my own styling, content, and extra features learned along the way.

## Features

- Responsive design — works on mobile, tablet, and desktop
- Background image hero section with dark overlay
- Logo swipe reveal animation on load
- Interactive service cards with hover effects
- Click-to-open modal with smooth open/close animation
- Stats section with key numbers
- Call to action banner
- Page loader that fades out on load

## Built With

- HTML5
- CSS3 (Flexbox, animations, media queries)
- Vanilla JavaScript (modals, loader)
- [Google Fonts — Poppins](https://fonts.google.com/specimen/Poppins)
- [Unsplash](https://unsplash.com/) — hero background image

## What I Learned

- How to structure a multi-section landing page
- CSS Flexbox for layouts and alignment
- Using `@keyframes` for CSS animations
- `max-width` and `overflow: hidden` for reveal animations
- CSS `transition` for smooth hover and modal effects
- JavaScript DOM manipulation for modal open/close
- `requestAnimationFrame` for triggering CSS transitions via JS
- Media queries for responsive design
- Git version control and pushing to GitHub
